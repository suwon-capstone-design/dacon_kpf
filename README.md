# 📰 신문과방송 독자 데이터 분석 솔루션
'신문과방송' 독자 데이터를 분석하여 콘텐츠 전략 인사이트를 제공하는 인터랙티브 웹 대시보드입니다.



## 🛠 주요 기능

- 지표 분석: 댓글, 좋아요, 조회수 가중치를 사용자가 직접 설정하여 상위 기사 랭킹 산출

- 독자층 분석: 기사별 주 소비층(연령/성별) 히트맵 및 누적 막대 그래프 시각화

- NLP 분석: Kiwi 형태소 분석기를 활용한 워드클라우드 및 LDA 토픽 모델링

- AI 전략 제언: 데이터 기반의 핵심 타겟 및 콘텐츠 전략 자동 멘트 제공



## 🚀 실행 방법
### 필수 라이브러리 설치
`pip install streamlit pandas matplotlib seaborn wordcloud kiwipiepy scikit-learn pyarrow`

### 앱 실행
`streamlit run main.py`
